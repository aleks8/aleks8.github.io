Medieval Warfare iOS Game

The project objective was to create a carefully planned game that had to satisfy certain practical expectations:
1. User Interface Assistance, the software must help the player control the game in an efficient way
2. Distribution and Game Server, multiple player functionality
3. Flexibility in Game Set Up, choosing amongst several "Islands" when starting the game
4. Saving, any player should be able to save the game and continue playing from there later

  • Role: Design & Development
  • Client: Professor Jorg Kienzle
  • Agency: Civet Atelier
  • Year: 2014 to 2015
